# SQL_Project
SQL_Project
Insurance Claims Analysis – MySQL Project

This project focuses on analyzing insurance claim data using MySQL. The database includes customer details, policy information, claim amounts, approval status, and fraud indicators.

Key features:

Designed relational database schema with primary and foreign keys

Performed advanced SQL queries (JOINs, GROUP BY, subqueries, aggregate functions)

Analyzed claim trends, approval rates, and high-risk customers

Identified potential fraudulent claims using data analysis techniques

The project demonstrates strong SQL skills in data extraction, transformation, and analytical reporting.
